Data Availability

Due to the large size of the data, we have restricted options to be able to download the summary statistics for all traits. We have the following options:

  • Colocalization and Rare Variant Analysis Results: You can download all colocalizing and rare variant analysis results for any given trait, gene, or SNP using the Download Data buttons of their respective pages.
  • GWAS / Summary Statistics: You can download all colocalizing and rare variant analysis results and their corresponding GWAS summary satistics only for a specific SNP, using the Download Data button on the SNP view.
  • Google Cloud Bucket: You can download all of the data from the Google Cloud Bucket. Note that this bucket is a 'requester pays' bucket, so you will need to have a Google Cloud account, be logged in, and have a credit card on file to download the data. A full explanation of the data in the bucket can be found here.

    Known Issues

    • Indel SNPs: Due to a bug in the data processing pipeline, indel SNPs were imputed, as opposed to the original SNPs being used. Due to high correlation between imputation and the original SNPs, this did not affect the colocalisaiton results.
